# DocsAgent

## speech_2_text
You need to install ffmpeg in the following directory: speech_2_text
[ffmpeg](https://www.gyan.dev/ffmpeg/builds/ffmpeg-release-essentials.7z)  

Make sure the directory structure should be as follows:  
speech_2_text    
└── ffmpeg  
	└── bin  
		├── ffmpeg.exe  
		├── ffplay.exe  
		└── ffprobe.exe
